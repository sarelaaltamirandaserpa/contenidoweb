Estructuras algorítmicas con JavaScript

Material para educación y práctica docente — actividades interactivas listas para el aula

Esta página muestra ejemplos claros y actividades prácticas para enseñar las estructuras algorítmicas (secuencia, selección y repetición) usando JavaScript. Incluye: explicaciones, ejemplos interactivos, ejercicios para estudiantes y notas pedagógicas para el docente.

Objetivos

  • Reconocer y aplicar las estructuras algorítmicas básicas: secuencia, selección y repetición.
  • Diseñar pequeñas actividades prácticas para estudiantes de secundaria y primeros semestres universitarios.
  • Relacionar la lógica algorítmica con situaciones reales en el aula.

2) Selección (decisiones: if / switch)

La selección permite que un algoritmo tome decisiones. Ejemplo: clasificar el promedio en una escala (A, B, C, D).

Ejemplo interactivo: Clasificación por promedio 
// Ejemplo de selección
function clasificar(promedio){
  if(promedio >= 4.5) return 'A';
  else if(promedio >= 3.5) return 'B';
  else if(promedio >= 2.5) return 'C';
  else return 'D';
}

3) Repetición (bucles: for, while, do-while)

Los bucles permiten repetir acciones. Ejemplo: sumar los números pares menores que N.

Ejemplo interactivo: Suma de pares 
// Ejemplo con for
function sumaPares(n){
  let suma = 0;
  for(let i=0; i<=n; i++){
    if(i % 2 === 0) suma += i;
  }
  return suma;
}

Actividades para el aula

  1. Mini-lab (20–30 min): En parejas, los estudiantes implementan la calculadora de promedio y la clasifican. Entregar captura de pantalla y pseudocódigo.
  2. Reto (30–40 min): Crear un programa que reciba edades de un grupo y calcule la moda, la media y la mediana. Discutir en clase las decisiones algorítmicas.
  3. Evaluación formativa: Pequeñas tareas con retroalimentación inmediata: completar tablas de ejecución (trace table) para algoritmos dados.

Notas pedagógicas

  • Usar ejemplos cercanos al estudiante (notas, listas de música, asistentes en el colegio) para motivar.
  • Combinar pseudocódigo y código real: primero pensar la lógica en lenguaje natural, luego programarla.
  • Promover la lectura de trazas (trace tables) para que los estudiantes entiendan cómo cambian las variables.

Resumen rápido

Secuencia, selección y repetición son las piezas básicas para el pensamiento algorítmico. Este recurso integra teoría, interacción y propuestas prácticas para docentes.

Recursos didácticos

  • Trace tables (plantilla)
  • Guía de preguntas socráticas para la evaluación
  • Plantilla de rúbrica editable

Consejos para el docente

  • Comenzar con ejemplos sin código (diagrama de flujo/pseudocódigo).
  • Mostrar errores habituales: índices fuera de rango, bucles infinitos, condiciones mal formuladas.
  • Fomentar la colaboración y la explicación entre pares.

Atajos rápidos (JS)

// forEach
[1,2,3].forEach(x => console.log(x));
// filter
[1,2,3,4].filter(x => x%2===0);
// map
[1,2,3].map(x => x*2);
Versión corta: este recurso es ideal para una sesión práctica (45–60 min) donde los estudiantes codifiquen y discutan sus soluciones.
